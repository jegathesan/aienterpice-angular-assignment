Note: During when i was working this assignment the given mock url https://run.mocky.io/v3/ca987707-db46-419e-9178-2bbdde3c3762 is not working and return CORS issue. So i've created the mock response file and used the mock json to rendere the shopping cart. Also i'm used dummy images
  • Before runnig application please install node_modules by running command npm i

  • Switch off the mock api mode. Change mockApi as false in environment.ts file. If mockApi is enabled first will look for mockdata from assets if mockdata is not found in assets service will fallback real server

  • Folder Structure

    • /app

      • /interceptors (MockData Interceptor for developement)

      • /modules (Application business modules)

      • /pages (Application pages)

      • /services (Includes data providers, Mock data file reference, and Response Interfaces)

      • /shared (Includes app reusable components, Helper classes, Rest end point config, Angular shared module)

        • /components (Includes app reusable components)

        • /helpers (API Helpers)

        • config.ts (API Endpoints)

      • /theme (Includes bootstrap override logic and CSS variables for global css theme config)

      • /assets (Includes static image, font, js files and mock data files)

        • /mockdata (Includes mock data files)
          • cart-items.json
